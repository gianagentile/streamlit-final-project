# MarketMetrics: Campaign Performance Dashboard

## Giana Gentile 

## App Description 
My app's intended use is to help users analyze and track the performance of marketing campaign. This is done by utilizing key metrics and interactive visualizations. In the app, users will be able to explore/examine campgain data, identify trends, and determine the effectiveness of the campgain based on click through rates (CTR) and conversion rates. 

## Intended Users 
This app is ultimaelty intended for marketing students, analyics students, and business who want to understand and evaluate the performace of marketing campagins. 

## Planned Features 
- Feature 1: Interactive Filtering of Data - here users can choose what data they want to see instead of eveything at once. It allows them to select specific campagins to narrow down the data
- Feature 2: Visualization Factor of Key Metrics - the visualizaton factor will visualize the impression, clicks, and conversions of a campagin. It basically turns numbers into visuals usuing charts and graphs.
- Feature 3: Automatic caculation of Metrics - this feature will provide automatic calculations of metrics of click through rate (CTR) and conversion rate by using formulas.
